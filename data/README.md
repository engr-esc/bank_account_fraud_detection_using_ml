# Data Folder

This project uses the Bank Account Fraud Dataset Suite - NeurIPS 2022.

Recommended source: https://www.kaggle.com/datasets/sgpjesus/bank-account-fraud-dataset-neurips-2022

The raw CSV files are not included in this repository because they are large. The notebook retrieves the dataset through `kagglehub`. If you download the files manually, place them in `data/raw/`.
